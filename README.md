PASTIMES: SECOND-HAND CLOTHING E-STOREPastimes is a user-friendly web application designed to bridge the gap between buyers and sellers of pre-owned clothing. The platform focuses on sustainability and making the trade of second-hand fashion accessible, safe, and efficient.🚀 PROJECT OVERVIEWThe goal of Pastimes is to create a seamless e-commerce experience specifically tailored for the South African "thrift" market. Users can declutter their wardrobes by selling items or find unique fashion pieces at affordable prices.✨ CORE FEATURESUSER AUTHENTICATION: Secure sign-up and login for both buyers and sellers.PRODUCT MANAGEMENT: Sellers can upload images, descriptions, sizes, and pricing for their clothing items.SEARCH & FILTER: Buyers can browse items by category (e.g., Vintage, Streetwear), size, and price.TRADING SYSTEM: Integrated functionality to facilitate the buying and selling process.RESPONSIVE DESIGN: Optimized for both desktop and mobile browsing using CSS.🛠️ TECH STACKThis project is built using the following technologies:FRONTEND: HTML5, CSS3, JavaScript (Bootstrap).BACKEND: PHP (Server-side logic).DATABASE: MySQL (Relational database for users and inventory).SERVER: XAMPP / Apache.📁 PROJECT STRUCTURE/pastimes
├── /css                # Stylesheets (Main.css)
├── /js                 # JavaScript (Validation.js)
├── /images             # Product and UI images
├── /includes           # Database connection (db.php)
├── index.php           # Homepage
├── shop.php            # Product listing page
└── dashboard.php       # User profile and seller management
⚙️ HOW TO RUN LOCALLYCLONE THE REPO:git clone https://github.com/YOUR_USERNAME/Pastimes.gitMOVE TO HTDOCS:Place the folder in C:/xampp/htdocs/.START XAMPP:Start Apache and MySQL from the control panel.DATABASE SETUP:Import the .sql file into phpMyAdmin.LAUNCH:Navigate to http://localhost/pastimes in your browser.🔮 FUTURE ENHANCEMENTSPAYMENT GATEWAY: Integration with local services like PayFast or Ozow.RATING SYSTEM: Peer-to-peer reviews to build community trust.CHAT SYSTEM: In-app messaging for buyers and sellers to negotiate.
