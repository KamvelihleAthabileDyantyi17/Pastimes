# 🛍️ Pastimes: Second-Hand Clothing E-Store

Pastimes is a user-friendly web application designed to bridge the gap between buyers and sellers of pre-owned clothing. The platform prioritizes sustainability and makes the trade of second-hand fashion accessible, safe, and efficient.

---

## 🚀 Project Overview

Pastimes creates a seamless e-commerce experience specifically tailored for the South African "thrift" market. Users can declutter their wardrobes by selling items or find unique fashion pieces at affordable prices.

---

## ✨ Core Features

- **User Authentication**: Secure sign-up and login for both buyers and sellers
- **Product Management**: Sellers can upload images, descriptions, sizes, and pricing for clothing items
- **Search & Filter**: Buyers can browse items by category (Vintage, Streetwear, etc.), size, and price
- **Trading System**: Integrated functionality to facilitate the buying and selling process
- **Responsive Design**: Optimized for desktop and mobile browsing

## ⚙️ How to Run Locally

### 1. Clone the Repository
```bash
git clone https://github.com/KamvelihleAthabileDyantyi17/Pastimes.git
```

### 2. Move to HTDOCS
Place the folder in `C:/xampp/htdocs/`

### 3. Start XAMPP
Start Apache and MySQL from the XAMPP control panel

### 4. Database Setup
Import the `.sql` file into phpMyAdmin

### 5. Launch
Navigate to `http://localhost/pastimes` in your browser
